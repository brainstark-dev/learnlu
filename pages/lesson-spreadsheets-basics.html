Spreadsheets for Beginners: Excel & Google Sheets

Spreadsheets are incredibly powerful tools for organising numbers, making budgets, and tracking anything. Excel and Google Sheets look intimidating at first — but the basics are simpler than they appear. This lesson starts from zero.

What is a Spreadsheet?

A spreadsheet is a program that organises data into a grid of rows and columns. You can type numbers, do automatic calculations, create charts, and track anything — budgets, inventory, grades, schedules. The two most popular tools are Microsoft Excel (paid) and Google Sheets (free).

A spreadsheet is like a super-powered calculator combined with a table. Imagine writing a budget on paper — you'd write numbers in rows and columns and manually add them up. A spreadsheet does all the adding (and any other maths) automatically, instantly recalculating every time you change a number.

Parts of a Spreadsheet

1

Cells

The individual boxes in the grid. Each cell has a unique address — like B3 (column B, row 3). Click a cell to select it. Type to enter data. Press Enter to move down, Tab to move right.

2

Rows (numbered)

Horizontal lines of cells, numbered 1, 2, 3... downwards. Row 1 is typically used for column headers ("Name", "Amount", "Date").

3

Columns (lettered)

Vertical lines of cells, labelled A, B, C... across. Each column holds one type of data (e.g. column A = names, column B = amounts).

4

Formula Bar

The bar just above the grid. It shows the contents of the currently selected cell — either the raw value or the formula calculating it.

5

Sheets (tabs at bottom)

A single spreadsheet file can contain multiple sheets — like pages in a notebook. Click the tabs at the bottom (Sheet1, Sheet2) to switch between them.

Basic Formulas — The Real Power

A formula is an instruction that tells the spreadsheet to calculate something. All formulas start with the = sign.

=A1+B1 # Add the values in cells A1 and B1 =A1-B1 # Subtract B1 from A1 =A1*B1 # Multiply A1 by B1 =A1/B1 # Divide A1 by B1 =SUM(A1:A10) # Add all values from A1 to A10 =AVERAGE(B1:B12) # Calculate the average of B1 through B12 =MAX(C1:C20) # Find the highest value in C1 to C20 =MIN(C1:C20) # Find the lowest value in C1 to C20

Monthly budget example: Put your income in cell B2. Put each expense in B3, B4, B5 etc. In cell B10, type =SUM(B3:B9) to total your expenses automatically. In B11, type =B2-B10 to show what's left. Now if you change any expense, everything recalculates instantly. This is the power of spreadsheets.

Formatting Cells

✓
Currency format: Select cells with money values → Home → $ button. Numbers display as ₦5,000.00 or $5,000.00 automatically
✓
Bold headers: Select Row 1 → Ctrl+B. Makes column headers stand out clearly
✓
Freeze top row: View → Freeze → 1 row. The header row stays visible as you scroll down through hundreds of rows
✓
Auto-fit column width: Double-click the border between column headers — it auto-sizes to fit the content
