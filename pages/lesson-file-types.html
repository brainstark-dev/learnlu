File Types and Extensions Explained — PDF, JPG, MP3 and More

What do file extensions like .pdf, .jpg, .mp4, .docx actually mean? Learn the most common file types and what programs open them.

What is a File Extension?

A file extension is the short code at the end of a file name, after the dot. It tells the operating system and users what type of data the file contains and which program should open it. photo.jpg — the extension is ".jpg" which means it's a JPEG image file.

🧠 Analogy

File extensions are like the labels on food containers. A jar labelled "COFFEE" tells you what's inside before you open it. A file labelled "report.pdf" tells you it's a PDF document before you click it. The label also tells you which "tool" to use — just as coffee needs a kettle, a PDF needs a PDF reader.

Document File Types

  • .docx — Microsoft Word document. Editable. Most common document format in offices.
  • .pdf — Portable Document Format. Looks identical on every device/printer. Not easily editable. Best for sharing final documents, CVs, invoices.
  • .txt — Plain text. No formatting. Opens in Notepad. Very small file size.
  • .xlsx — Microsoft Excel spreadsheet. Contains rows, columns, and formulas.
  • .pptx — Microsoft PowerPoint presentation. Contains slides.

Image File Types

  • .jpg / .jpeg — Most common photo format. Compressed. Smaller file size. Slight quality loss. Used for photographs.
  • .png — Higher quality than JPG. Supports transparent backgrounds. Used for logos and graphics.
  • .gif — Supports animation. Used for short looping animations (GIF memes).
  • .webp — Modern format from Google. Small file size with good quality. Used on websites.

Audio and Video File Types

  • .mp3 — Most common audio format. Compressed music file. Plays in almost everything.
  • .mp4 — Most common video format. Good quality at reasonable file size. Plays everywhere.
  • .wav — Uncompressed audio. Very high quality but large file size. Used in music production.
  • .mkv — High-quality video container. Supports multiple audio tracks and subtitles. Popular for movies.
✅ Sending a CV

Always save and send your CV as a .pdf file, not a .docx. A PDF looks exactly the same on the employer's computer as it does on yours. A .docx may look different if the employer has a different version of Word, ruining your formatting.

📌 Key Takeaways

  • File extension = the code after the dot that identifies the file type
  • .docx = Word document; .xlsx = Excel; .pptx = PowerPoint; .pdf = portable document
  • .jpg = compressed photo; .png = higher quality with transparency support
  • .mp3 = audio; .mp4 = video
  • Always send CVs and important documents as .pdf — consistent appearance on all devices
