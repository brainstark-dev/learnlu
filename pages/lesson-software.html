What is Software? Apps, Programs & the Difference Explained

You use software every time you open an app, browse a website, or type a document. But do you know what software actually is, or how it's different from hardware? This lesson explains everything.

What is Software?

Software is a set of instructions that tells a computer what to do. It is everything on a computer that you cannot physically touch — apps, programs, games, and the operating system itself. Without software, hardware is completely useless.

Hardware is like a piano — a physical instrument. Software is like sheet music — the instructions that tell you what notes to play. The piano alone makes no music. The sheet music alone makes no sound. Together, they create something beautiful. Your computer (hardware) needs software to do anything useful.

The Three Types of Software

1

System Software

The foundational layer. This includes the operating system (Windows, macOS, Android) and drivers — small programs that help hardware like printers and graphics cards communicate with the OS. You rarely interact with this directly.

2

Application Software (Apps)

The programs you actually use every day — Microsoft Word, Google Chrome, WhatsApp, YouTube, Excel, Zoom. These are built on top of the system software and rely on the OS to function.

3

Utility Software

Tools that maintain and optimise your system — antivirus programs, disk cleanup tools, backup software, and file compression tools (like WinRAR). They work behind the scenes to keep things running smoothly.

Apps vs Programs — What's the Difference?

The terms "app" and "program" mean essentially the same thing — both are pieces of software you run on a device. Historically, "program" was the desktop term and "app" (short for application) referred to mobile software. Today, the words are used interchangeably.

Microsoft Word on your laptop = a program (or desktop app). WhatsApp on your phone = a mobile app. WhatsApp Web in your browser = a web app. All three are software. The difference is only where they run — on your desktop OS, your mobile OS, or inside a browser.

Never download software from unknown websites. Fake versions of popular software (like "free Microsoft Office") are the #1 way viruses get onto computers. Only download from official websites or trusted app stores (Google Play Store, Apple App Store, Microsoft Store).

Installing Software on Windows

1

Go to the official website

Always download from the software maker's official website. For Chrome: google.com/chrome. For Zoom: zoom.us. Never search "free download Chrome" — you might land on a fake site.

2

Download the installer (.exe file)

Click the download button. A file ending in .exe (Windows) or .dmg (Mac) will download to your Downloads folder.

3

Double-click and follow the steps

Open the downloaded file and click through the installation wizard. Read each screen — untick any unwanted "bonus" toolbars or software that might try to sneak in.

4

Launch the program

Once installed, the program appears in your Start Menu and possibly on your Desktop. Double-click to open it.
