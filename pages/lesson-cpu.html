CPU Explained: The Brain of Your Computer

The CPU does all the thinking in your computer. Every click, every calculation, every game frame — the CPU handles it all. This lesson explains what it is, how it works, and what to look for when buying one.

What is a CPU?

CPU stands for Central Processing Unit. It is the main chip inside your computer responsible for carrying out every instruction from every program you run. When you open Chrome, type a document, or play a video, the CPU is doing the actual computational work.

The CPU is like the chef in a restaurant kitchen. Customers (you) send orders (commands). The chef reads each order, prepares the dish (processes the data), and sends it to the table (output). The faster the chef works, the faster you get your food — and the more orders they can handle at once. A faster CPU means a snappier, more responsive computer.

How a CPU Works — The Fetch-Decode-Execute Cycle

Every CPU follows the same basic cycle billions of times per second:

1

Fetch

The CPU retrieves the next instruction from RAM — for example, "add these two numbers."

2

Decode

The CPU figures out what the instruction means and what it needs to do to carry it out.

3

Execute

The CPU carries out the instruction — does the maths, moves the data, or triggers an output.

When you press Ctrl+S to save a document: (1) CPU fetches the "save" instruction. (2) It decodes: "write data to storage." (3) It executes: copies the document from RAM to your hard drive. This happens in a fraction of a second.

CPU Speed — What Does GHz Mean?

CPU speed is measured in GHz (Gigahertz). 1 GHz = 1 billion cycles per second. A 3.5 GHz CPU performs 3.5 billion fetch-decode-execute cycles every second.

GHz is like the tempo of a metronome. A faster tempo (higher GHz) means the CPU can process more beats (instructions) per second. A 4.0 GHz CPU ticks 4,000,000,000 times per second — that's 4 billion tiny calculations happening while you blink.

CPU Cores — What Are They?

A core is an independent processing unit within the CPU chip. A dual-core CPU has 2 cores (2 independent processors on one chip). A quad-core has 4. Each core can work on a different task simultaneously.

Core CountNameBest For
2 coresDual-coreBasic tasks: browsing, email, simple documents
4 coresQuad-coreEveryday use, video calls, light photo editing
6–8 coresHexa/Octa-coreGaming, multitasking, video editing
12–24 coresHigh-endProfessional video production, 3D rendering, servers

Cores are like the number of chefs in the kitchen. One chef (single-core) can only cook one dish at a time. Four chefs (quad-core) can cook four dishes simultaneously. You can have 20 browser tabs open and each core handles some of them — that's why more cores = better multitasking.

Intel vs AMD — The Two Main CPU Brands

BrandPopular ModelsKnown For
IntelCore i3, i5, i7, i9Strong single-core performance, widely compatible, common in laptops
AMDRyzen 3, 5, 7, 9Often better value for money, strong multi-core performance

For most beginners: an Intel Core i5 or AMD Ryzen 5 is the sweet spot — fast enough for everything everyday, affordable, and available in most mid-range laptops.
