Basic Computer Troubleshooting — Fix Problems Yourself

Most computer problems have simple fixes that anyone can do. This lesson teaches you how to diagnose and solve the most common issues — saving you time and money.

Rule #1 — Try Restarting First

Before anything else: restart the computer. This solves a surprisingly large number of problems — slow performance, frozen apps, WiFi issues, update problems, and more. Restarting clears the RAM, closes stuck processes, and gives the system a fresh start.

A restart is like a good night's sleep for your computer. After hours of running dozens of apps simultaneously, the computer accumulates temporary files, memory leaks, and stuck processes — like a person whose desk gets cluttered throughout the day. Sleep clears the desk completely, and they wake up fresh and focused. Restart your computer at least once a week.

Common Problems and How to Fix Them

ProblemLikely CauseFix
Computer is frozenApp crashed or RAM fullWait 30 seconds, then: Ctrl+Alt+Delete → Task Manager → End task for the frozen app. If that fails, hold power button 10 seconds to force shutdown.
Very slow performanceToo many startup programs, full storage, or malwareRestart; disable startup programs (Task Manager → Startup tab); free up storage; run Windows Defender scan
Won't connect to internetRouter issue or WiFi adapter problemRestart router (unplug 30 sec); run Windows Troubleshooter (right-click WiFi icon → Troubleshoot)
App won't openCorrupted installation or missing fileRestart computer; uninstall and reinstall the app
Blue Screen of Death (BSOD)Driver error or hardware issueNote the error code; restart; if recurring, search the error code online for specific fix
Computer overheatingDusty vents, intense tasks, or failing fanEnsure vents are not blocked; clean dust from vents with a soft brush; use on hard surface, not a bed

Using Task Manager

Task Manager is one of the most useful tools on any Windows computer. It shows you what's running, what's using your CPU/RAM, and lets you force-close frozen apps.

1

Open Task Manager

Press Ctrl+Alt+Delete → click "Task Manager". Or press Ctrl+Shift+Esc directly.

2

Find the problem app

In the Processes tab, look for any app using very high CPU (%) or Memory (%). An app saying "Not Responding" is frozen.

3

End the task

Right-click the app → "End task". This force-closes it. Warning: any unsaved work in that app will be lost.

When to Get Professional Help

⚠️
The hard drive is making clicking or grinding noises — possible failure, back up data immediately
⚠️
Repeated Blue Screens with different error codes — hardware issue needing diagnosis
⚠️
Laptop screen cracked or completely dark — physical repair needed
⚠️
Computer won't turn on at all — power supply or hardware issue

📌 Troubleshooting order: (1) Restart (2) Check internet connection (3) Task Manager (4) Restart router (5) Run Windows Update (6) Run virus scan (7) Search the error message online (8) Call a technician
