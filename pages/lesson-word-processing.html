Word Processing for Beginners: Microsoft Word & Google Docs

Word processors let you create professional documents — CVs, letters, reports, and assignments. This lesson teaches you everything you need to know about Microsoft Word and Google Docs, including formatting, saving, and printing.

What is Word Processing?

Word processing is using software to create, edit, format, and print text documents. The two most popular tools are Microsoft Word (paid, industry standard) and Google Docs (free, online, autosaves). Both can open and save the same .docx file format.

A word processor is a typewriter with superpowers. Like a typewriter, you type text. But unlike a typewriter, you can move text anywhere, change its appearance instantly, undo mistakes, check spelling automatically, add images, and save multiple versions — all before printing a perfect final copy.

The Word Interface — What You're Looking At

1

The Ribbon (Top)

The toolbar at the top containing all formatting options. Organised into tabs: Home (text formatting), Insert (images, tables), Layout (page size, margins), Review (spell check). Click each tab to see its tools.

2

The Document Area (Centre)

The white page where you type. The blinking cursor (|) shows where your next character will appear. Click anywhere in the document to move the cursor there.

3

The Status Bar (Bottom)

Shows page count, word count, and zoom level. Right-click it to add more information like character count or reading time.

Saving Your Document — Critical Habits

The biggest mistake beginners make: not saving regularly. Word loses unsaved work when it crashes or the power goes out. Save every 5–10 minutes as a habit: press Ctrl + S. It takes less than one second.

1

Save for the first time (Ctrl+S or Ctrl+Shift+S)

Choose a location (Documents folder), type a meaningful filename ("Job Application CV — Ada 2025"), and click Save. Avoid names like "Document1" that you'll never find again.

2

Save As — Create a Copy

File → Save As saves a duplicate with a new name. Use this for version control: "CV v1", "CV v2 — Updated", "CV Final". Never overwrite your only good version.

3

Enable AutoSave (Word 365)

In Microsoft 365/OneDrive, enable the AutoSave toggle (top-left) to save to the cloud every few seconds automatically — you'll never lose work again.

Writing a CV? Use Google Docs — it's free and autosaves, so you won't lose your work. Applying for a job that asks for a .docx file? In Google Docs: File → Download → Microsoft Word (.docx). Your Google Doc is instantly saved as a Word-compatible file, ready to attach to your email.

Printing Your Document

1

Press Ctrl + P

Opens the Print dialog in any program.

2

Choose your printer

Select from the available printers. If your printer isn't showing, check it's turned on and connected (USB or WiFi).

3

Check settings

Number of copies, page range (print all or specific pages), orientation (portrait for normal documents, landscape for tables), and paper size (A4 for most countries, Letter for the US).

4

Click Print

Your document is sent to the printer queue and will print shortly.
