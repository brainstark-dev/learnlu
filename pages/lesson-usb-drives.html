USB Drives and External Storage — Complete Guide

USB drives (also called flash drives, thumb drives, or memory sticks) are tiny devices that let you carry files in your pocket. This lesson explains how to use them correctly and safely.

What is a USB Drive?

A USB drive (also called a flash drive, thumb drive, or memory stick) is a small portable storage device that plugs into your computer's USB port. It uses the same flash memory technology as an SSD — no moving parts, very durable, and holds data even when unplugged.

A USB drive is like a tiny portable filing cabinet you can carry in your pocket. When you plug it into a computer, it's like opening that filing cabinet on that computer's desk. You can copy files in or out. When you unplug it, you take the filing cabinet with you — and the computer no longer has access to those files.

How to Use a USB Drive

1

Plug it in

Insert the USB drive into any USB port on your computer. After a few seconds, it appears as a new drive in File Explorer (usually labelled D:, E:, or F:).

2

Copy files to the USB drive

Find the file you want to copy. Right-click → Copy (or Ctrl+C). Open the USB drive in File Explorer. Right-click empty space → Paste (or Ctrl+V). The file is now on the USB drive.

3

Safely eject before removing

IMPORTANT: Click the USB icon in the taskbar (bottom right) → "Eject" the drive. Wait for "Safe to remove hardware" message. Then physically remove the USB. This prevents data corruption.

Never just yank a USB drive out while files are copying or while the drive is being accessed. The drive's internal file system can become corrupted, making your files unreadable. Always use "Safely Remove Hardware" first — it only takes one extra second and protects your data.

Using External Drives for Backup

The best practice for protecting your important files is the 3-2-1 backup rule:

✓
Keep 3 copies of important data
✓
On 2 different types of storage (e.g., computer hard drive + external drive)
✓
With 1 copy offsite (e.g., in the cloud via Google Drive)

A journalist lost 5 years of interview recordings when her laptop was stolen. She had no backup. Replacing those recordings was impossible — the original interview subjects were no longer available. One external hard drive ($30–50) or Google Drive subscription would have saved everything. Don't wait for disaster to start backing up.

USB Safety Warning

Never plug in a USB drive you found on the street or received from an unknown person. In 2010, the US military network was infected by malware spread through infected USB drives that were deliberately left in car parks near military bases. Employees picked them up and plugged them in out of curiosity. One unknown USB can compromise your entire computer and spread to any network it connects to.
