Files & Folders: Organising Your Computer Like a Pro

Files are where your data lives. Folders are how you organise it. Understanding this system is essential for using any computer confidently — and it's much simpler than most people think.

What is a File?

A file is a collection of data stored on your computer under a single name. Every photo, document, song, video, and program on your computer is a file. Files are the basic building blocks of all digital information.

A file is like a single sheet of paper. It has a name (the file name), content (the data inside), and a type (what kind of information it holds). Just like paper, you can create new ones, edit them, copy them, move them somewhere else, or throw them away.

File Names and Extensions

Every file has a name and an extension — the few letters after the dot that tell the OS what type of file it is. For example: holiday.jpg → the name is "holiday", the extension .jpg means it's a photo.

What is a Folder?

A folder (also called a directory) is a container for files. You can put files and other folders inside a folder to keep things organised. Folders can be nested — a folder inside a folder inside a folder — creating a hierarchy.

Your computer's storage is like a large filing cabinet. Folders are the dividers inside the cabinet — labelled "Work", "Personal", "Photos 2024". Files are the individual sheets of paper inside each section. A well-organised filing cabinet means you can find anything in seconds. A disorganised one means hours of searching.

Windows File Explorer — Your File Manager

File Explorer is the program on Windows for browsing all your files and folders. Open it by pressing Windows + E or clicking the folder icon on the taskbar.

1

Left panel — Navigation

Shows your main locations: Desktop, Documents, Downloads, Pictures, This PC. Click any to jump straight to it.

2

Right panel — Contents

Shows all files and folders in the location you selected. Double-click any folder to open it. Double-click a file to open it in its program.

3

Address bar at top

Shows exactly where you are: C:\Users\Ada\Documents\Work. Click any part of it to jump to that level.

You have 200 photos from your holiday in your Downloads folder. Good organisation: Create a folder called "Photos" in Documents. Inside it, create "2025 Lagos Trip". Move all 200 photos into it. Now in 2 years, when you want to find them: Documents → Photos → 2025 Lagos Trip. Done in 5 seconds, not 20 minutes of searching.

The Recycle Bin — Your Safety Net

When you delete a file, it doesn't disappear — it moves to the Recycle Bin. This is your safety net. To restore: open Recycle Bin, right-click the file, choose Restore. To permanently delete everything: right-click the Recycle Bin icon → Empty Recycle Bin.

Accidentally deleted an important file? Don't panic — check the Recycle Bin first. It will almost certainly still be there. Only files deleted with Shift+Delete bypass the Recycle Bin and are permanently gone immediately.
