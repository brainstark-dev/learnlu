How to Print from a Computer: Complete Beginner's Guide

Printing seems simple — but there are several settings beginners miss that lead to wasted paper, wrong sizes, or print jobs stuck in a queue. This lesson walks through everything from connecting a printer to printing your first document perfectly.

Connecting a Printer to Your Computer

Printers connect to computers in two ways: via a USB cable (plug in and Windows installs it automatically) or via WiFi (printer and computer on the same WiFi network). WiFi printing is more convenient — you never need a cable.

1

USB Connection

Turn the printer on. Plug the USB cable from printer to computer. Windows detects it and installs the driver automatically within 1–2 minutes. Done.

2

WiFi Connection

On the printer: press WiFi/Wireless button → follow screen to select your WiFi network → enter password. On Windows: Settings → Bluetooth & devices → Printers & scanners → Add a printer. Windows finds and adds it automatically.

The Print Dialog — All Settings Explained

Press Ctrl + P in any program to open the Print dialog. Here's what each option does:

SettingWhat It ControlsCommon Choice
PrinterWhich printer to useSelect your printer from the list
CopiesHow many copies to print1 (default)
PagesPrint all or specific pages"All" or type "1-3" for pages 1 to 3
OrientationPortrait (tall) or Landscape (wide)Portrait for documents, Landscape for tables/photos
Paper sizeA4, A3, Letter etc.A4 for most countries, Letter for USA
ColourColour or black & white (grayscale)Grayscale to save ink for text documents
Print PreviewSee exactly how the page will look before printingAlways check this before clicking Print

Always check Print Preview before clicking Print. This prevents wasting paper on documents with wrong margins, missing content, or the wrong orientation. In Google Docs and Chrome, the preview appears automatically in the print dialog.

Common Printing Problems & Fixes

1

Nothing prints — job stuck in queue

Start Menu → search "printers" → open your printer → right-click any stuck job → Cancel. Then try printing again. Restart the printer if needed.

2

Printer not found

Check: Is the printer turned on? Is it connected (USB or WiFi)? For WiFi: Is the printer on the same WiFi network as your computer? Try turning the printer off and on.

3

Faint or streaky text

Low ink or toner. On the printer, run the "head cleaning" or "nozzle check" from the printer's menu. If that fails, the ink cartridge needs replacing.

4

Paper jam

Turn off the printer. Open all access panels and gently pull out the jammed paper — always pull in the direction of paper travel. Never force it. Close all panels before turning back on.

Printing to PDF — No Printer Needed

You can "print" a document to a PDF file instead of paper — creating a digital copy with perfect formatting. In any Print dialog: select "Microsoft Print to PDF" or "Save as PDF" as the printer. Click Print, choose a save location, and a PDF file is created.

You need to send someone a copy of your invoice but don't want them to edit it. Type it in Word, then Ctrl+P → Print to PDF. The resulting PDF looks exactly like the printed version — all formatting preserved — and can be emailed to anyone. They can view it but not easily edit it.
