Windows 11 for Beginners: Desktop, Taskbar & Start Menu

Windows is the operating system on most computers in the world. This lesson walks you through everything you see when Windows starts — the desktop, taskbar, Start Menu, and how to find and use your apps.

The Desktop

When Windows starts up, the first thing you see is the Desktop — a large background area where you can place shortcuts to your most-used files and programs. Think of it as your virtual office desk — keep the things you use most on top.

The Desktop is like the top of a physical desk. You put your most important items there — your pen pot, notebook, and phone. Everything else lives in drawers and cabinets (your folders and drives). A clean desk helps you work faster. A desktop with 200 random icons is as chaotic as a cluttered physical desk.

✓
Double-click any icon to open it
✓
Right-click any icon for options: Open, Delete, Rename, Properties
✓
Right-click the Desktop (empty area) to change wallpaper, display settings, or create new folders
✓
Press Windows + D to instantly show the Desktop (minimises all open windows)

The Taskbar

The Taskbar is the horizontal bar at the bottom of the screen. It shows your pinned apps, currently open programs, and system information.

Taskbar AreaWhat's There
Start button ⊞ (Centre)Opens the Start Menu to find and launch apps
Search barType anything to search your computer, files, or the web
Pinned appsYour favourite apps for quick access — drag to reorder, right-click to remove
Open appsPrograms currently running — click to switch between them
System tray (right side)Clock, date, WiFi signal, volume, battery level, notification bell

To pin an app to the Taskbar: Find the app in the Start Menu, right-click it, and choose "Pin to taskbar." It will appear in the Taskbar permanently for one-click access. This is how you set up your personal quick-launch area.

The Start Menu

Press the Windows key ⊞ or click the Start button to open the Start Menu. This is where you find every app, setting, and file on your computer.

1

Pinned Apps Section (Top)

Your most-used apps displayed as large icons. Right-click any to unpin it. Click "All apps" to see every installed program alphabetically.

2

Search Bar (Top of Start Menu)

Type any app name to find it instantly. Type "word" to find Microsoft Word. Type "settings" to jump to Windows Settings. Fastest way to launch anything.

3

Recommended Section (Bottom)

Shows recently opened files and newly installed apps. Useful for quickly reopening something you worked on yesterday.

4

Power Button (Bottom Right)

Sleep, Shut down, or Restart. Always use Shut Down through the Start Menu — pulling the power cord can corrupt files.

Managing Windows (The Program Windows)

Each program opens in its own "window" — a rectangular frame on screen. You can resize, move, minimise, or close any window.

ButtonSymbolWhat It Does
Minimise— (dash)Hides the window to the Taskbar — program is still running
Maximise□ (square)Makes the window fill the entire screen
Restore⧉ (two squares)Returns a maximised window to its previous size
Close× (cross)Closes the window — program stops running

Snap feature: Drag a window to the left edge of the screen — it snaps to fill exactly half the screen. Drag another window to the right edge — it fills the other half. Now you can see two programs side by side. This is the best way to work with two documents or a browser and a document at the same time.

Windows Settings

Access all your computer's settings through Windows Settings. Press Windows + I to open it directly.

✓
System — Display brightness, sound volume, notifications, power settings
✓
Bluetooth & devices — Connect mice, keyboards, headphones, printers
✓
Network & internet — Connect to WiFi, see data usage
✓
Personalisation — Change wallpaper, colours, themes, Start Menu layout
✓
Windows Update — Check for and install security updates
