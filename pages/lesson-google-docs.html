Google Docs Complete Guide for Beginners

The free alternative to Microsoft Word that works anywhere. Create, share, and collaborate in seconds.

What Is Google Docs?

Google Docs is a free online word processor — like Microsoft Word but in your web browser with no installation, no cost, and automatic saving. Everything is stored in Google Drive. Access it at docs.google.com with any Google account.

Creating Your First Document

1

Go to docs.google.com

Sign in with your Google account. Click the large + (Blank) to create a new document.

2

Name your document

Click "Untitled document" at the top and type a name. It saves automatically.

3

Start typing

Click in the white page area and type. Every keystroke is saved to Google Drive automatically — you never need to press Ctrl+S.

Google Docs autosaves every few seconds and keeps a full version history. You can see every change ever made: File → Version History → See version history. Click any past version to restore it. You can never permanently lose your work.

Formatting Text in Google Docs

All the same formatting as Word works in Google Docs: Bold (Ctrl+B), Italic (Ctrl+I), Underline (Ctrl+U), font size, font type, alignment, bullet lists, numbered lists, and headings.

To make a professional-looking heading: type your title, select it, change the size to 18–20pt, make it Bold, and use the "Styles" dropdown (showing "Normal Text" by default) to set it as "Heading 1". Google Docs then uses this for an automatic table of contents.

Sharing and Collaborating

Google Docs' superpower is real-time collaboration. Multiple people can edit the same document simultaneously — you see their cursor and changes appear instantly.

1

Click "Share" (top right)

A dialog appears. Enter the email addresses of people you want to share with.

2

Set permission level

Editor = can change the document. Commenter = can leave comments only. Viewer = can only read.

3

Or share via link

Click "Copy link" to get a URL anyone with the link can access — useful for sharing with many people.

Working Offline

✓
Install the Google Docs Offline Chrome extension
✓
In Google Drive Settings, enable "Offline" mode
✓
You can now edit docs without internet — changes sync when you reconnect
