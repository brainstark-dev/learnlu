Microsoft Word for Beginners: Write Your First Document

Open Word, type a document, format it beautifully, save it, and print it — all explained from scratch.

📸 Real Photos 🎨 Visual Aids

Opening Microsoft Word

Microsoft Word is the world's most popular word processor — used for letters, CVs, essays, reports, and any typed document. To open it: press the Windows key, type "Word," and click Microsoft Word when it appears.

Microsoft Word is a digital typewriter with superpowers. Unlike a real typewriter, you can fix mistakes instantly, change the font and size, move paragraphs around, add images, check spelling automatically, and print a perfect copy every time.

Don't have Microsoft Word installed? Use Google Docs (docs.google.com) — it's completely free, works in your browser with no installation, and can save files in Word format (.docx).

Typing and Formatting Text

Click anywhere on the blank white page and start typing. The blinking cursor shows where your text will appear. Press Enter to start a new paragraph. Press Backspace to delete the character behind the cursor.

To format your name as a large, centred heading: click at the top of the page, type your name, select all the text of your name (click and drag), change the font size to 18 or 20, click the Centre alignment button, and make it Bold. Now it looks like a proper document header.

Saving Your Document

1

Press Ctrl + S to save

The first time, a "Save As" dialog opens asking where to save the file and what to name it.

2

Choose a location

Select Documents for work files. Give your file a clear name: "Job Application Letter" not "Document1".

3

Press Ctrl + S regularly while working

Every 5–10 minutes while working. If your computer crashes, you only lose the last few minutes of work.

Power outages are common in Nigeria. Save your work every few minutes with Ctrl+S. Better yet, enable AutoSave by clicking File → Options → Save → check "AutoSave" and set interval to 1–2 minutes.

Useful Word Features

✓
Spell check: Wavy red underlines = spelling errors. Right-click for correction suggestions.
✓
Word count: View → Word Count, or look at the bottom status bar
✓
Undo: Ctrl+Z undoes your last action — you can undo many steps back
✓
Find & Replace: Ctrl+H — replace every instance of a word at once
✓
Print: Ctrl+P — opens the print dialog
